## 2. ระบบนี้ทำอะไร (ขอบเขตงาน)

ระบบเดียวครบ 2 ส่วน:

**ส่วน A — แปลผลแลป + เปรียบเทียบรายปี**
- ทีมงานอัปโหลดผลแลป (รูปถ่าย/ไฟล์ PDF) → AI อ่านและแปลผลอัตโนมัติ
- แสดงผลเป็นหน้าเว็บ + ออกรายงาน PDF (มีโลโก้/ใบอนุญาตโรงพยาบาล, QR ตรวจสอบของจริง, หมอเซ็นรับรอง, 2 ภาษา)
- เก็บค่าทุกตัวลงฐานข้อมูล → เปรียบเทียบแนวโน้มข้ามปีเป็น **กราฟ**
- คะแนนสุขภาพรวม + ระบบสี (เขียว/เหลือง/แดง) เข้าใจง่ายใน 2 วินาที
- แนะนำตรวจเพิ่ม/บริการต่อยอด (upsell) ตามค่าที่ผิดปกติ

**ส่วน B — CRM ตามขายข้าม Pillar อัตโนมัติ**
- รวมลูกค้าคนเดียวให้เห็นทุกบริการที่เคยใช้ (ข้าม 7 pillar)
- ตามขายอัตโนมัติผ่าน LINE / Messenger / Instagram
- เมื่อลูกค้า add LINE → ระบบไล่ส่งข้อความตามขายเป็นสเต็ป หยุดเมื่อลูกค้าตอบ/จองแล้ว
- กระดานไปป์ไลน์ (Kanban) ติดตามสถานะลูกค้า

### ค่าเริ่มต้น (จ่ายครั้งเดียว): **~฿0 – 500**

## 8. ข้อดีเชิงคุณภาพของ Claude Code (นอกจากราคา)

- **เร็ว:** เริ่มใช้งานได้ในหลักวัน–สัปดาห์ ไม่ต้องรอ 3–4 เดือน → เก็บลูกค้า/รายได้เร็วขึ้น
- **แก้ไข/ต่อยอดทันที:** อยากเพิ่มฟีเจอร์ ปรับ copy ปรับกราฟ → สั่งได้ทันที ไม่ต้องรอคิว dev หรือจ้างรอบใหม่
- **ไม่มีความเสี่ยงเรื่องคน:** ไม่มีปัญหา dev ลาออกกลางทาง ส่งงานช้า หรือเก็บความรู้ไว้คนเดียว
- **ความรู้ในระบบอยู่ครบ:** เอกสาร/แผน/โค้ดอยู่ในรีโปเดียว สืบทอดต่อได้

## 10. ตัวคุมต้นทุน run รายเดือน
1. **Prompt caching** — ลดค่า AI บน input ได้ ~90%
2. **Push LINE แบบเจาะกลุ่ม** (เฉพาะลูกค้าค่าผิดปกติ/มีแนวโน้มซื้อ) ไม่หว่าน broadcast → คุมโควตา LINE
3. **เริ่มจาก free tier** Supabase/Vercel ก่อน อัปเกรดเมื่อปริมาณโต
