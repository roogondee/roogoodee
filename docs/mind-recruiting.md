# Mind Pillar — Recruiting Plan (In-House Experts)

แผน recruit นักจิตวิทยา/จิตแพทย์ in-house สำหรับ Phase 2 ของ `mind` pillar
**Goal**: 2-3 นักจิตวิทยาคลินิก + 1 จิตแพทย์ที่ปรึกษา ภายใน 60 วัน

## Why in-house (ไม่ใช่ partner platform)

ตัดสินใน PR #82:
- Margin per session 100% (vs. 50-70% ถ้าผ่าน Ooca/iSTRONG)
- Brand consistency (ผู้ใช้เห็น Roogondee ทั้ง funnel)
- Pattern alignment กับ pillar อื่น (ทุก pillar = in-network กับ W Medical)
- Data ownership 100%

Trade-off: ใช้เวลา recruit 1-3 เดือน — ระหว่างนั้น `mind` อยู่ใน Phase 1 waitlist mode

## Target Roles

### Role 1: นักจิตวิทยาคลินิก (Clinical Psychologist) × 2-3 คน

**Profile**:
- ปริญญาโทขึ้นไป สาขาจิตวิทยาคลินิก/Counseling Psychology
- มี **ใบอนุญาตประกอบวิชาชีพจิตวิทยาคลินิก** (พ.ร.บ. 2562) — required
- ประสบการณ์ 2+ ปี ทำ individual / couples therapy
- เคยทำ telehealth (Zoom/Doxy.me/Healthbird) — preferred
- พูดไทย คล่อง + อังกฤษ (สำหรับรับ expat) — nice to have
- เปิดรับ LGBTQ+ clients — required

**Work model**:
- Freelance / contract (ไม่ใช่พนักงานประจำ)
- Session 30-50 นาที, telehealth (video/phone)
- รับเคสตามตารางที่กำหนดเอง (minimum 4 sessions/สัปดาห์, max 20)
- Roogondee นัดหมาย + จัดการ payment / scheduling system

**Compensation** (เสนอตัวเลือก):
- **Option A** — fixed per session: **1,500-2,500฿/session** (เราเก็บ margin จากการ scale lead pipeline)
- **Option B** — rev-share: 70% to therapist, 30% to Roogondee per paid follow-up session (intro 30-min ฟรียังเป็นของ provider 100%)

แนะนำ Option A สำหรับ provider ใหม่ + Option B สำหรับ provider ที่อยากเติบโตกับเรา

**Specialty preference** (จัด pool ให้ครอบคลุม):
- mood & anxiety (most common demand)
- relationships / couples
- grief & loss
- LGBTQ+ affirmative
- trauma / PTSD (advanced, optional)

### Role 2: จิตแพทย์ที่ปรึกษา (Consultant Psychiatrist) × 1 คน

**Profile**:
- แพทยศาสตร์ + วุฒิบัตรจิตเวชศาสตร์ (specialty in Psychiatry)
- มีใบอนุญาตประกอบวิชาชีพเวชกรรม — required
- ประสบการณ์ 5+ ปี
- เปิดรับเคส telehealth + สั่งยาผ่าน e-prescription

**Work model**:
- Consultant on-call (เคส refer มาจากนักจิตวิทยา)
- Session 45 นาที, telehealth
- ดูแลเคสที่ต้องประเมิน medication หรือ severe cases
- 2-5 sessions/สัปดาห์

**Compensation**:
- **2,500-4,000฿/session** (depending on case complexity)

## Interview Rubric (เก็บคะแนน 5 มิติ × 5 pt = 25 pt)

| มิติ | คะแนน | Indicator |
|---|---|---|
| **Clinical competence** | /5 | ตอบ case study ได้แม่น, อ้างอิงทฤษฎี (CBT/ACT/DBT), discuss risk assessment ได้ |
| **Empathy & presence** | /5 | warmth, active listening, non-judgmental tone ใน interview |
| **Boundary awareness** | /5 | เข้าใจ scope of practice, รู้เมื่อต้อง refer out, ปฏิบัติตาม code of ethics |
| **Telehealth readiness** | /5 | คล่อง tech, มี private space, ใช้ tool ได้ |
| **Brand fit** | /5 | สื่อสารด้วย tone "ส่วนตัว ไม่ตัดสิน", ต้นทุน relationship/LGBTQ-friendly, สอดคล้องกับ Roogondee voice |

**Threshold**: ≥18/25 = ผ่าน + sign contract

## Interview Process

```
Application → screen (CV + license check) → Stage 1 → Stage 2 → Offer

Stage 1 (45 min, online):
  - Intro to Roogondee model
  - Case study walk-through (provided 24h before)
  - Q&A

Stage 2 (60 min, online):
  - Role-play 1 mock session (Roogondee staff = simulated client)
  - Crisis scenario test (self-harm disclosure mid-session — how respond?)
  - Logistics (rate, hours, start date)

Offer:
  - Sign contract (NDA + service agreement)
  - Onboarding (next section)
```

## Onboarding Checklist (Week 1)

- [ ] Sign service agreement + NDA
- [ ] License verification (ตรวจสอบกับ สมาคมจิตวิทยาคลินิก หรือแพทยสภา)
- [ ] Bank account / KYC
- [ ] Provider profile setup (photo, bio, specialties, languages)
- [ ] Tech setup: Cal.com booking, Zoom Pro link, Doxy.me account (optional)
- [ ] Read & sign acknowledgement: Crisis SOP (`docs/mind-crisis-sop.md`)
- [ ] Read & sign: Roogondee brand voice guide (TBD)
- [ ] 1 supervised session (shadow Roogondee senior or first session reviewed)
- [ ] First live session scheduled

## Legal / Compliance

- **Service agreement** (template TBD)
