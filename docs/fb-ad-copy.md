## วิธีใช้

### Copy-paste ลง Boost Post
1. Copy caption ทั้ง block → paste ใน post composer
2. ดูตัวอย่าง (preview) ว่าลิงก์เป็น clickable card หรือไม่
3. ถ้าลิงก์ไม่ขึ้นเป็น card → ลบ http:// แล้วใส่ใหม่ (FB cache บางครั้ง)
4. โพสต์ → รอ 4-6 ชม. ดู organic baseline → ค่อย Boost

### Copy-paste ลง Ads Manager
1. Campaign → Ad → Primary Text: paste caption (ตัด URL ออก)
2. Website URL: paste link แยก
3. Headline (40 char): copy บรรทัดแรก
4. Description (30 char): copy บรรทัด CTA

### Variant testing
ทุก angle ยิงคู่กัน → หลัง 7 วันเลือก angle ที่ CPL ต่ำสุด แล้ว scale งบ
