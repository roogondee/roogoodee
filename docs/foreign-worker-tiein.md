## ตรวจสุขภาพเพื่อต่อ Work Permit — รายละเอียด 9 ข้อ

1. **6 โรคต้องห้ามที่ต้องคัดกรอง**
   - โรคเรื้อน (Leprosy)
   - วัณโรคระยะอันตราย (Tuberculosis)
   - โรคเท้าช้าง (Elephantiasis)
   - โรคติดยาเสพติดให้โทษ (Drug Addiction)
   - โรคพิษสุราเรื้อรัง (Chronic Alcoholism)
   - ซิฟิลิสระยะที่ 3 (Tertiary Syphilis)
2. **เอกสารที่ต้องเตรียม**
   - หนังสือเดินทาง (Passport) — สำเนา/ตัวจริง
3. **เอกสารนายจ้าง** — สำเนาบัตรประชาชน หรือ หนังสือรับรองบริษัทของนายจ้าง
4. **แบบฟอร์มใบรับรองแพทย์** — โรงพยาบาลเตรียมไว้ให้
5. **สถานที่ตรวจ** — ต้องเป็นโรงพยาบาล/คลินิกที่ได้รับการรับรอง — เริ่มต้น **500 บาท**
6. **ระยะเวลารอผล** — ประมาณ 1.5–2 ชั่วโมง (ตรวจหมู่ ≥ 50 คน อาจรับใบวันรุ่งขึ้น)
7. **อายุใบรับรองแพทย์** — ใช้ได้ไม่เกิน 90 วันนับจากวันตรวจ
8. **การเตรียมตัว** — ไม่ต้องงดน้ำ/อาหาร (ยกเว้นบางแพ็กเกจที่รวมการตรวจอื่น)
9. **ภาษา** — โรงพยาบาลที่มีบริการสื่อสารภาษาเมียนมา / จีน / อังกฤษ

## ขั้นตอนการตรวจสุขภาพแรงงานต่างด้าว — 7 ขั้นตอน

ลำดับการตรวจที่ W Medical (ใช้เป็น process flow สำหรับ FB post / blog / story):

1. **ลงทะเบียน + ตรวจเอกสาร** — หนังสือเดินทาง (Passport) + เอกสารนายจ้าง
2. **ตรวจสุขภาพทั่วไป** — ชั่งน้ำหนัก / วัดส่วนสูง / ความดันโลหิต / ชีพจร โดยแพทย์
3. **ตรวจคัดกรอง 6 โรคต้องห้าม** — เรื้อน, วัณโรคระยะอันตราย, เท้าช้าง, ติดยาเสพติด, พิษสุราเรื้อรัง, ซิฟิลิสระยะที่ 3
4. **ตรวจห้องปฏิบัติการ** — ปัสสาวะ + เลือด (ห้องแล็บมาตรฐาน MOPH LAB)
5. **เอกซเรย์ปอด** — Chest X-ray คัดกรองวัณโรค
6. **สแกนม่านตา + Facial Recognition (Iris Scan)** — ยืนยันตัวตนตามมาตรฐานกรมควบคุมโรค สำหรับแรงงานข้ามชาติ (ประกาศนียบัตรอบรม อธิบดีกรมควบคุมโรค 3 ม.ค. 2568)
7. **รับใบรับรองแพทย์** — รอผลประมาณ 1.5–2 ชั่วโมง (กลุ่ม ≥ 50 คน อาจรับใบวันรุ่งขึ้น)

> หมายเหตุ: ขั้นตอนที่ 6 สแกนม่านตา/ใบหน้า เป็นการยืนยันตัวตนตามระเบียบใหม่ของกรมควบคุมโรคสำหรับแรงงานข้ามชาติ — รพ. ที่ทำได้ต้องมีบุคลากรผ่านการอบรมและได้รับประกาศนียบัตรเท่านั้น

## Usage notes (สำหรับ content gen)

- เมื่อเขียน blog/FB post `service: 'foreign'` ให้แทรก provider credentials block ในส่วน "ทำไมต้องตรวจกับ รพ. ที่ได้รับอนุญาต" หรือ FAQ
- ใช้ "ขั้นตอนการตรวจสุขภาพแรงงานต่างด้าว — 7 ขั้นตอน" ด้านบนเป็น canonical process flow — **ต้องมีขั้นตอนสแกนม่านตา (Iris Scan)** เสมอ
- Caption สั้น (FB story / IG): ใช้ headline `"รพ. ที่ได้รับอนุญาตตรวจสุขภาพแรงงานต่างด้าว"` + คำว่า `มาตรฐาน MOPH LAB` เพื่อสร้างความน่าเชื่อถือ (ในสื่อลูกค้าให้ใช้คำว่า "มาตรฐาน MOPH LAB" แทนเลขใบรับรอง — เลข LA 7044P/2568 เก็บไว้เป็นหลักฐานภายในในตาราง credentials ด้านบนเท่านั้น)
