# Myanmar Community Clinic Flyer — Context Pack

Source material สำหรับทำ **ใบปลิวประชาสัมพันธ์บริการรักษาทั่วไป** ของโรงพยาบาล
เจาะกลุ่ม **แรงงานเมียนมา (พม่า)** ในสมุทรสาคร — เน้นบริการ OPD ทั่วไป
(ฉีดยา / ทำแผล / เจ็บป่วยทั่วไป) ไม่ใช่ชุดตรวจสุขภาพต่อ Work Permit
(ชุดนั้นอยู่ที่ `docs/foreign-worker-tiein.md` — คนละใบปลิว)

> เป้าหมายใบปลิว: ให้แรงงานพม่ารู้ว่า "เจ็บป่วยเล็กน้อยก็มาได้ มีคนพูดพม่าได้ ราคาไม่แพง ใกล้บ้าน"
> ลดความกลัวเรื่องภาษา/เอกสาร/ค่าใช้จ่าย เพื่อให้กล้าเดินเข้ามาใช้บริการ

## Provider — โรงพยาบาลทั่วไปขนาดเล็กดับเบิ้ลยู เมดิคอล (W Medical)

- **ที่อยู่:** 99/26 หมู่ 5 ต.บางน้ำจืด อ.เมืองสมุทรสาคร จ.สมุทรสาคร 74000
- **โทร:** 0 2453 6121
- **จุดขายหลัก:** มีเจ้าหน้าที่/ล่าม **สื่อสารภาษาเมียนมาได้** + ใบอนุญาตสถานพยาบาลถูกต้อง
- **เลขที่ใบอนุญาต:** สถานพยาบาล (สมุทรสาคร) 001/2569 — *ห้ามแก้เลข เป็นข้อมูลราชการ*

## บริการที่จะโปรโมตในใบปลิว (general OPD)

| บริการ (ไทย) | Burmese | English |
|---|---|---|
| ฉีดยา / ฉีดวัคซีน | ဆေးထိုး / ကာကွယ်ဆေးထိုး | Injection / Vaccination |
| ทำแผล / เย็บแผล | အနာဖာ ပြုစု / အနာချုပ် | Wound dressing / Suturing |
| ตรวจรักษาโรคทั่วไป (ไข้ ไอ ปวด ท้องเสีย) | ယေဘုယျ ရောဂါ ကုသမှု (အဖျား၊ ချောင်းဆိုး၊ နာကျင်၊ ဝမ်းလျှော) | General illness (fever, cough, pain, diarrhea) |
| น้ำเกลือ / ให้น้ำเกลือ | ဆားရည် တင်ခြင်း | IV fluid / saline drip |
| จ่ายยา | ဆေးဝါး | Pharmacy / dispensing |
| ใบรับรองแพทย์ | ဆရာဝန် ထောက်ခံစာ | Medical certificate |

## เลย์เอาต์ใบปลิวที่แนะนำ (A5 แนวตั้ง พิมพ์ 2 ภาษา)

1. **หัวกระดาษ** — โลโก้ W Medical + รู้ก่อนดี + headline 2 ภาษา (ไทยบน พม่าล่าง)
2. **แถบไอคอนบริการ** — 6 ไอคอน (ฉีดยา/ทำแผล/รักษาทั่วไป/น้ำเกลือ/จ่ายยา/ใบรับรอง) พร้อมคำ 2 ภาษาใต้ไอคอน
3. **กล่องจุดเชื่อใจ** — 3 ✅ (พูดพม่าได้ / ใบอนุญาตถูกต้อง / ราคาเป็นกันเอง)
4. **แผนที่ + ที่อยู่ + เบอร์โทร + QR LINE @roogondee**
5. **ฟุตเตอร์** — เวลาทำการ (ขอจากพันธมิตร) + ภาษาที่ให้บริการ: เมียนมา / ไทย / อังกฤษ
