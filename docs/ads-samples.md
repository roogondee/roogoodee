# Creative Production Checklist

ส่งให้ designer ทำ:

| Service | Image 1:1 (1080) | Image 1.91:1 (1200×628) | Reel 9:16 (15s) |
|---|---|---|---|
| GLP-1 (3 angle) | 3 ชิ้น | 3 ชิ้น | 1-2 ชิ้น |
| STD (3 angle) | 3 ชิ้น | 3 ชิ้น | 1 ชิ้น (privacy-led) |
| CKD (3 angle) | 3 ชิ้น | 3 ชิ้น | 1 ชิ้น (family-led) |

**Brand specs:**
- Primary mint `#52B788`, base forest `#1B4332`
- Logo "รู้ก่อนดี" ขวาล่างทุกชิ้น
- โลโก้ W Medical Hospital ขวาบน (ถ้ามี partner agreement)
- Font: Sarabun / Noto Sans Thai

**Avoid (Meta will reject):**
- Before/after photos
- Personal pronouns "คุณ" + body imagery
- Numerical claims ("ลด 10 กิโลใน 30 วัน")
- Syringe / needle / blood imagery (STD)
- Distressed/scared faces

# Tracking

ทุก ad ยิง URL form:
```
https://roogondee.com/quiz/{service}?utm_source=facebook&utm_medium=cpc&utm_campaign={campaign_id}&utm_content={ad_id}
```

Quiz funnel ยิง events:
- `PageView` (auto)
- `quiz_start` (custom)
- `quiz_complete` (custom)
- `Lead` (standard, value=score, currency=THB) ← optimize นี่
- `CompleteRegistration` (standard, backup)

# Approval Workflow

ก่อน publish ทุก ad ผ่าน checklist:

- [ ] Primary text ≤ 125 chars
- [ ] Headline ≤ 40 chars
- [ ] Description ≤ 30 chars
- [ ] ไม่มี policy violation (ดู checklist ใน `docs/fb-ads-brief.md` ส่วน 6)
- [ ] UTM ครบ (source, medium, campaign, content)
- [ ] Landing URL HTTPS + working
- [ ] Pixel ติดบน landing (เช็คด้วย Pixel Helper extension)
- [ ] Creative ผ่าน brand guideline (mint/forest, no before/after, no body imagery)
